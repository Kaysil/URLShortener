# ShortURL PLUGIN #
### ShortURL plugin based on ShortURLAPI for C3CBot
### Author: Kaysil (me) ##

### Usage/Using:
- Put it to ./plugins folder then run your bot
#### Command:
- Main command: /short

#### Example:
- /short <url> [vanity]<br>
- /short https://google.com<br>
  - /short https://google.com ditmegoogle
