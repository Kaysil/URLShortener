# URLShortener PLUGIN #
### URLShortener plugin based on URLShortenerAPI for C3CBot
### Author: Kaysil (me) ##

### Usage/Using:
- Put it to ./plugins folder then run your bot
#### Command:
- Main command: /short

#### Example:
- /short (url) (vanity)
```
- /short https://google.com
```
